# Movie Review Sentiment Analysis

Classifies the sentiment of movie reviews (positive / negative) using NLP text
preprocessing and a classical machine-learning classifier.

## Files

| File | Purpose |
|------|---------|
| `sentiment-analysis.ipynb` | Text preprocessing + model training/eval notebook |
| `moviereviews.csv`         | Labelled review text |

## Run

```bash
pip install -r ../requirements.txt
jupyter notebook sentiment-analysis.ipynb
```
