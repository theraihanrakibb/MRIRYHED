# DBSCAN Customer Segmentation

Segments customers by their spending behaviour using **DBSCAN** (Density-Based
Spatial Clustering of Applications with Noise) — a clustering algorithm that finds
arbitrarily shaped clusters and flags outliers, unlike K-Means.

## Files

| File | Purpose |
|------|---------|
| `dbscan-customer-segmentation.ipynb` | Clustering + evaluation notebook |
| `wholesome_customers_data.csv`        | Customer expenditure features |

## Run

```bash
pip install -r ../requirements.txt
jupyter notebook dbscan-customer-segmentation.ipynb
```
