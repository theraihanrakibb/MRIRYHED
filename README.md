# Mini Projects 🧩

A consolidated showcase of small beginner/learning projects, merged from several
older repositories into one place. Each folder is an independent mini-project.

## Projects

| Folder | What it is | How to run |
|--------|------------|------------|
| `zodiac/` | Zodiac sign calculator (single-page app). | Open `zodiac/index.html` in a browser. |
| `Weather-App/` | Weather app with icons (HTML/CSS/JS). | Open `Weather-App/index.html` in a browser. |
| `Online-CodeEditor/` | In-browser code editor (HTML/CSS/JS). | Open `Online-CodeEditor/index.html` in a browser. |
| `Real-Time-Chat-App/` | Real-time chat app — source archived as `backend.zip` + `frontend.rar`. | Extract the archives to run. |
| `ML-Project/` | Collection of ML mini-projects (notebooks + datasets): heart-disease prediction, house-price regression, KNN (sonar), DBSCAN clustering, SVM wine fraud, sentiment analysis, country clustering, recommender system. | Open the `.ipynb` notebooks with Jupyter. |

## Notes
- These are early learning projects kept here for the record; they are not
  representative of current work.
- `Real-Time-Chat-App` only contains archived source (`backend.zip` / `frontend.rar`)
  from the original repository.
